## 📌 About

This is a premium, single-page portfolio CV built with modern web technologies. It presents my professional profile as a BCA student passionate about Cybersecurity, Cloud Computing, and Full Stack Development. The design features smooth animations, interactive elements, and a glass-morphism UI that creates an immersive user experience.

- ✅ **Fully Responsive** – Optimized for all devices (desktop, tablet, mobile)

## 🚦 Getting Started

### 1. Clone the Repository
```bash
git clone https://github.com/Kumarshubham2007/portfolio-cv.git
cd portfolio-cv
```

### 2. Open in Browser
- Simply open `index.html` in your preferred browser.
- Or use Live Server in VS Code for hot-reload.

### 3. Customize
- **Profile Image:** Click on the profile circle to upload a new photo.
- **Skills:** Modify the `skills` array in the `<script>` section.
- **Projects/Experience:** Update the HTML sections directly.
- **Colors:** Edit the `:root` CSS variables for primary/secondary colors.

## 🧩 Key Features Explained

### Circular Skill Bars
Each skill is represented by a circular progress bar that animates when scrolled into view. The percentage is calculated dynamically from the `skills` data array.

### Glass-morphism Cards
Cards have a semi-transparent background with backdrop blur, creating a modern frosted-glass effect. They also feature a subtle 3D tilt on hover.

### Particle Network
A lightweight canvas-based particle system that creates a dynamic, connecting-dot background effect with mouse-responsive glow.

### Scroll Progress
A thin gradient bar at the top of the page indicates how far the user has scrolled, enhancing navigation UX.
